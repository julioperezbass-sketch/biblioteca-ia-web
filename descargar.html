Instrucciones para Mac

  1. Descarga el archivo DMG.
  2. Abre el DMG con doble clic.
  3. Arrastra la app a Aplicaciones.
  4. Abre Canciones de Adoracion desde Aplicaciones.

Si macOS muestra un aviso de seguridad, haz clic derecho sobre la app y pulsa Abrir.

Instrucciones para Windows

  1. Descarga el archivo zip.
  2. Descomprime la carpeta.
  3. Abre Canciones de Adoracion.exe.

Si Windows SmartScreen muestra un aviso, pulsa Mas informacion y luego Ejecutar de todos modos.
